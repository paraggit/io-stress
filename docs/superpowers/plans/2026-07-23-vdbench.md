# Vdbench Tool Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add selectable Vdbench Phase-1 workloads via typed `tools.vdbench` config and `--tool`, generating param files and exec’ing `vdbench` in-pod while auto-skipping lifecycle.

**Architecture:** Typed `config.VDBench` holds block/filesystem patterns. `pkg/vdbench` builds param files. Workload branches on `tools.active`: FIO path unchanged; vdbench path writes params into the pod and runs `vdbench -f … -o …`. Pods use the active tool image; container name becomes `iotool`.

**Tech Stack:** Go 1.26, cobra, existing k8s exec, YAML config.

## Global Constraints

- Spec: `docs/superpowers/specs/2026-07-23-vdbench-design.md`
- Default `tools.active: fio` — no behavior change for existing users
- Vdbench image default: `quay.io/pakamble/vdbench:latest`
- Phase 1 only for vdbench; auto-skip Phase 2/3 when `active=vdbench`
- Mapping: block patterns → RBD Block; filesystem patterns → RBD FS + CephFS
- Skip CephFS RWX sub-phase for vdbench
- Container name: `iotool` for both engines
- Do not implement smallfiles or vdbench lifecycle

```go
func needsBlockPatterns(cfg *Config) bool {
	for i := 1; i <= cfg.Cluster.RBD.NumPVC; i++ {
		if i%2 == 0 {
			return true
		}
	}
	return false
}
func needsFilesystemPatterns(cfg *Config) bool {
	if cfg.Cluster.CephFS.NumPVC > 0 {
		return true
	}
	for i := 1; i <= cfg.Cluster.RBD.NumPVC; i++ {
		if i%2 == 1 {
			return true
		}
	}
	return false
}
```

```go
func TestBuildBlockParam(t *testing.T) {
	got := BuildBlockParam(
		config.VDBenchBlock{Size: "15g"},
		config.VDBenchPattern{Name: "random_write", Rdpct: 0, Seekpct: 100, Xfersize: "4k", Skew: 0},
		"/dev/rbdblock", 60,
	)
	want := "sd=sd1,lun=/dev/rbdblock,openflags=o_direct,size=15g\n" +
		"wd=wd1,sd=sd1,rdpct=0,seekpct=100,xfersize=4k,skew=0\n" +
		"rd=rd1,wd=wd1,elapsed=60,interval=1\n"
	if got != want {
		t.Fatalf("got:\n%s\nwant:\n%s", got, want)
	}
}

func TestBuildFilesystemParam(t *testing.T) {
	got := BuildFilesystemParam(
		config.VDBenchFilesystem{
			Depth: 4, Width: 5, Files: 10, FileSize: "1m",
			OpenFlags: "o_direct", GroupAllFWDsInOneRD: true,
		},
		config.VDBenchPattern{Name: "sequential_write", Rdpct: 0, Seekpct: 0, Xfersize: "1m", Skew: 0},
		60,
	)
	if !strings.Contains(got, "fsd=fsd1,anchor=/mnt/data,depth=4,width=5,files=10,size=1m,openflags=o_direct\n") {
		t.Fatalf("fsd line missing: %s", got)
	}
	if !strings.Contains(got, "group_all_fwds_in_one_rd=yes") {
		t.Fatalf("missing group flag: %s", got)
	}
}
```
